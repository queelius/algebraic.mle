# Method for obtaining the parameters of an \`mle_fit\` object.

Method for obtaining the parameters of an \`mle_fit\` object.

## Usage

``` r
# S3 method for class 'mle_fit'
params(x)
```

## Arguments

- x:

  the \`mle_fit\` object to obtain the parameters of

## Value

Numeric vector of parameter estimates.

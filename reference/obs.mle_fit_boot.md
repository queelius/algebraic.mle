# Method for obtaining the observations used by the \`mle_fit_boot\`.

Method for obtaining the observations used by the \`mle_fit_boot\`.

## Usage

``` r
# S3 method for class 'mle_fit_boot'
obs(x)
```

## Arguments

- x:

  the \`mle_fit_boot\` object to obtain the number of observations for

## Value

The original data used for bootstrapping.

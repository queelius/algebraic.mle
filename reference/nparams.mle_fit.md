# Method for obtaining the number of parameters of an \`mle_fit\` object.

Method for obtaining the number of parameters of an \`mle_fit\` object.

## Usage

``` r
# S3 method for class 'mle_fit'
nparams(x)
```

## Arguments

- x:

  the \`mle_fit\` object to obtain the number of parameters of

## Value

Integer number of parameters.

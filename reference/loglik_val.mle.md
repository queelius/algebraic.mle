# Method for obtaining the log-likelihood of an \`mle\` object.

Method for obtaining the log-likelihood of an \`mle\` object.

## Usage

``` r
# S3 method for class 'mle'
loglik_val(x, ...)
```

## Arguments

- x:

  the log-likelihood \`l\` evaluated at \`x\`, \`l(x)\`.

- ...:

  additional arguments to pass

## Value

the log-likelihood of the fitted mle object \`x\`

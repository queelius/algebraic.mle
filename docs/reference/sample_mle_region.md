# Function for obtaining sample points for an \`mle\` object that is within the \`p\`-probability region.

Function for obtaining sample points for an \`mle\` object that is
within the \`p\`-probability region.

## Usage

``` r
sample_mle_region(n, x, p = 0.95)
```

## Arguments

- n:

  the sample size

- x:

  the \`mle\` object

- p:

  the probability region

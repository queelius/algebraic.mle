# Computes the variance-covariance matrix of \`mle_emp\` object.

Computes the variance-covariance matrix of \`mle_emp\` object.

## Usage

``` r
# S3 method for mle_emp
vcov(x, ...)
```

## Arguments

- x:

  the \`mle_emp\` object to obtain the variance-covariance of

- ...:

  additional arguments to pass

# Computes the bias of an \`mle_normal_mu_var\` object.

Computes the bias of an \`mle_normal_mu_var\` object.

## Usage

``` r
# S3 method for mle_normal_mu_var
bias(x, par = NULL, ...)
```

## Arguments

- x:

  the \`mle\` object to compute the bias of.

- par:

  the true parameter value, defaults to unknown (NULL)

- ...:

  unused additional arguments

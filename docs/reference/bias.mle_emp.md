# Computes the estimate of the bias of a \`mle_emp\` object.

Computes the estimate of the bias of a \`mle_emp\` object.

## Usage

``` r
# S3 method for mle_emp
bias(x, par, ...)
```

## Arguments

- x:

  the \`mle_emp\` object to compute the bias of.

- par:

  The true parameter value.

- ...:

  pass additional arguments

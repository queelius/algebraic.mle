# General two sample hypothesis test.

General two sample hypothesis test.

## Usage

``` r
two.sample.test(sampler, obs, cdf = NULL, options = list(), ...)
```

## Arguments

- sampler:

  random sampler for reference distribution

- obs:

  a sample of observations

- cdf:

  a cdf for the reference distribution, defaults to NULL

- n:

  sample size (for sampler)

- method:

  the method for the two sample hypothesis test

## Value

hypothesis test object

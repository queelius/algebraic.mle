# Generic method for obtaining the log-likelihood of fitted distribution object, assuming it has a likelihood function, e.g., a fitted \`mle\` object.

Generic method for obtaining the log-likelihood of fitted distribution
object, assuming it has a likelihood function, e.g., a fitted \`mle\`
object.

## Usage

``` r
loglike(x, ...)
```

## Arguments

- x:

  the object to obtain the log-likelihood of

- ...:

  additional arguments to pass

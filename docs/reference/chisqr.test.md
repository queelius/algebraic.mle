# chi-squar goodness-of-fit test

chi-squar goodness-of-fit test

## Usage

``` r
chisqr.test(obs, cdf, options = list(), ...)
```

## Arguments

- obs:

  a sample of observations

- cdf:

  the cdf of the reference distribution

- options:

  a list of options to pass to the test function

- ...:

  additional arguments to pass

- breaks:

  the number of breaks to use in the histogram

- rescale.p:

  logical, whether to rescale the p-value

- simulate.p.value:

  logical, whether to simulate the p-value

- nbreaks:

  the number of breaks to use in the histogram

## Value

a hypothesis test object

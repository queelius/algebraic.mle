# Method for obtained the confidence interval of an \`mle_emp\` object.

Method for obtained the confidence interval of an \`mle_emp\` object.

## Usage

``` r
# S3 method for mle_emp
confint(x, level = 0.95, ...)
```

## Arguments

- x:

  the \`mle_emp\` object to obtain the confidence interval of

- level:

  the confidence level

- ...:

  additional arguments to pass
